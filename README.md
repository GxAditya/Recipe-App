# Recipe App

A mobile application built with Kotlin that allows users to browse and manage food recipes. This project demonstrates modern Android development practices, including a clean UI and efficient data handling.

## Features

- Browse a variety of recipes with high-quality images.
- View detailed information including ingredients and cooking instructions.
- Search functionality to find specific dishes quickly.
- Responsive design optimized for Android devices.

## Tech Stack

- Language: Kotlin
- Build System: Gradle (Kotlin DSL)
- IDE: Android Studio

## Getting Started

### Prerequisites

- Android Studio Flamingo or newer
- JDK 17 or higher
- Android SDK (API Level 24 or higher recommended)

### Installation

1. Clone the repository:
   ```bash
   git clone [https://github.com/GxAditya/Recipe-App.git](https://github.com/GxAditya/Recipe-App.git)
   ```

2. Open the project in Android Studio:
* File > Open > Select the cloned directory.


3. Sync the project with Gradle files.
4. Run the app:
* Connect an Android device or start an emulator.
* Click the "Run" button (green play icon) in Android Studio.



## Project Structure

* /app: Contains the main source code, resources, and UI components.
* /gradle: Configuration files for the Gradle build system.
* build.gradle.kts: Project-level build configuration.
* settings.gradle.kts: Module and repository settings.
